    Атака с малым индексом открытого ключа

    Условия атаки

    \(e\) очень мало, например \(e\) = 3.

    Принцип атаки

    Предположим, пользователь использует ключ \(e=3\). Учитывая, что отношение шифрования выполнено:

    \[c \equiv m^3 \bmod N\]

    Тогда:

    \[\begin{align*} m^3 &= c+k\times N \\ m &= \sqrt[3]{c+k \times n} \end{align*}\]

    Злоумышленник может перечислить \(k\) от мала до велика и по очереди открыть три корня, пока не будет открыто целое число.

    Пример

    Видя \(e=3\), ясно, что атаковали малый индекс открытого ключа. Здесь мы используем библиотеку Crypto для чтения открытого ключа и используем многопроцессорность для ускорения взлома.

    import gmpy2
from Crypto.PublicKey import RSA
from multiprocessing import Pool

pool = Pool(4)

with open('./pubkey.pem', 'r') as f:
    key = RSA.importKey(f)
    N = key.n
    e = key.e

with open('flag.enc', 'r') as f:
    cipher = f.read().encode('hex')
    cipher = int(cipher, 16)


def calc(j):
    print(j)
    a, b = gmpy2.iroot(cipher + j * N, 3)
    if b == 1:
        m = a
        print('{:x}'.format(int(m)).decode('hex'))
        pool.terminate()
        exit()


def small_e():
    inputs = range(0, 130000000)
    pool.map(calc, inputs)
    pool.close()
    pool.join()

small_e()

    Производный алгоритм RSA - алгоритм Рабина

    Условия атаки

    Алгоритм Рабина характеризуется \(e=2\).

    Принцип атаки

    Зашифрованный текст:

    \[c = m ^ 2 \bmod n\]

    Расшифровка:

    • Рассчитайте \(m_p\) и \(m_q\):

      \[\begin{align*} m_p &= \sqrt{c} \bmod p \\ m_q &= \sqrt {c} \bmod q \end{align*}\]

    • Рассчитайте \(y_p\) и \(y_q\) расширенным алгоритмом Евклидова:

      \[y_p \cdot p + y_q \cdot q = 1\]

    • Решите четыре открытых текста:

      \[\begin{align*} a &= (y_p \cdot p \cdot m_q + y_q \cdot q \cdot m_p) \bmod n \\ b &= n - a \\ c &= (y_p \cdot p \cdot m_q - y_q \cdot q \cdot m_p) \bmod n \\ d &= n - c \end{align*}\]

    Info

    Если \(p \equiv q \equiv 3 \pmod 4\), тогда

    \[\begin{align*} m_p &= c^{\frac{1}{4}(p + 1)} \bmod p \\ m_q &= c^{\frac{1}{4}(q + 1)} \bmod q \end{align*}\]

    В целом \(p \equiv q \equiv 3 \pmod 4\) достаточно, в случаях, которые не выполняются, обратитесь к соответствующему алгоритму.

    Напишем код

    import string
from Crypto.PublicKey import RSA
from Crypto.Util.number import inverse
# Read public key parameters
with open('pubkey.pem', 'r') as f:
    key = RSA.importKey(f)
    N = key.n
    e = key.e

with open('flag.enc', 'r') as f:
    cipher = f.read().encode('hex')
    cipher = string.atoi(cipher, base=16)

print("please input p")
p = int(raw_input(), 10)
print('please input q')
q = int(raw_input(), 10)

inv_p = inverse(p, q)
inv_q = inverse(q, p)

mp = pow(cipher, (p + 1) / 4, p)
mq = pow(cipher, (q + 1) / 4, q)

a = (inv_p * p * mq + inv_q * q * mp) % N
b = N - int(a)
c = (inv_p * p * mq - inv_q * q * mp) % N
d = N - int(c)

for i in (a, b, c, d):
    s = '%x' % i
    if len(s) % 2 != 0:
        s = '0' + s
    print(s.decode('hex'))
